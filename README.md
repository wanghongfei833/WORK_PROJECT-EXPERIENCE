## 项目简介

### 松材线虫病害检测
无人机航拍 + 语义分割（DeepLabv3+）检测松材线虫病，解决正样本稀少问题，使用 MixUp、Copy-Paste、CycleGAN 风格迁移等增强手段。

### 螺纹松动检测
针对两期螺纹图像进行松动角度计算，创新设计角点标注方案 + YOLO 检测 + 小型 CNN 角度回归。

### 隧道衬砌无损检测
2023.6–2025.9 大型工程化项目。轨道小车搭载 8K 面阵相机采集隧道表观影像，YOLOv5 检测裂缝/渗水/掉块，后处理含 C++ 重写的 BFS 宽度长度计算。

### 非接触式人体生理指标检测
双光相机（RGB + 红外）实现 5M 非接触式体温（0.2℃ 精度）和心率（MAE < 5）测量。涵盖 RPPG 模型选型（EfficientPhys）、单目测距、多模态温度偏置回归模型。
