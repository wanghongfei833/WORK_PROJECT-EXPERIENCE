# 项目背景

松材线虫病是一种针对松树的机严重病害，树木一旦感染就无法治疗，并且会大规模传染，因此需要进行防治，人工巡检难度比较大，通常采用无人机航飞然后在进行人工复核，主要工作就是替换人工复核的问题。

# 项目难点

1. **正样本稀少**：松材线虫感染的病害被发现后一般会立刻铲除，因此样本数据很少。
2. **背景复杂**：航拍图像的背景复杂，可能有各种情况导致模型误分。

# 工作内容

主要工作内容是数据处理、模型选型、模型训练与优化、数据输出。

## 数据处理

数据采集后，通常按照 512 大小进行裁剪，重叠按照 100 pix 进行，然后进行数据标记。

### 数据增强

由于正样本过少，因此项目中采取了许多的数据增强方案，除开常用的旋转翻转等，还加入 MixUp、CPOY-PAUSE 等方法，同时为了进一步增加数据量，利用风格迁移模型 CycleGAN 进行正常树木 → 病害数据的生成，这更好的丰富了数据量。

### 模型训练与优化

模型主要采用了 CNN 模型，横向对比尝试了 UNET（以及U2NET）、FCN、DeepLab 等，最后发现 DeepLab 的效果和速度比较突出，最后选用了 DeelLabv3+ 使用 resnet50 作为 Backbone。

针对小样本采取了以下方法进行优化：

1. 使用 FocalLoss+DiceLoss 结合替换交叉熵损失函数，提升模型对于困难样本的准确率。
2. 将正样本（带病害）和负样本（无病害）进行混合采样，也就是每个 batch 都混合有无病害数据。

### 数据导出

完成工作流：

```text
TIFF 数据 → 模型推理 → 结果 merge → shp 导出
```
